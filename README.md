# film_rails_app
website for keeping track of actors in film for directors. Users create an account or sign in through facebook and create film/actor pages
and review films they did not create.

to install, fork from https://github.com/elTuttle/film_rails_app, cd into film_rails_app and then cd into film, run bundle install and
rails s to start to start server with application
